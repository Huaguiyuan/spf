2. QUICK START

2.1. WHAT YOU NEED TO DO TO RUN THE PROGRAM

  • Download the latest version of the KONDO program.
  • Find out if your system has the required libraries (gsl, lapack, blas, etc). If not, download and compile them
  • Choose the Makefile that best resembles your system.
  • Edit it and select the location of the libraries (gsl, lapack, etc) if you installed them.
  • Select a model through a -DMODEL_KONDO_... define
  • Compile and link the program
  • Choose a sample input file and edit it
  • Run the program

4. RUN-TIME ISSUES

4.1 REQUIREMENTS

  • GSL, LAPACK and BLAS libraries.
  • PERL and the argument.pl file
  • MPI Library if running the parallel version
  • Some additional files maybe required depending on the type of the run

4.2 RUN-TIME INSTRUCTIONS

The program takes only one argument: the name of the input file. This is a unix-style configuration file where the format is: 
LABEL VALUE
Comments are specified by using the # sign at the beginning of the line. Inline comments are not allowed. Sample configuration files are provided within the KONDO package. One of these can be used and edited. Also see Section FIXME.

4.3 INTERACTIVE RUN

Use: 
./name_of_the_executable name_of_the_input_file

4.4 PBS RUN

Serial run script: 
#!/bin/ksh 
#PBS -N test
#PBS -q batch
#PBS -l walltime=12:00:00,ncpus=1
cd your_directory/
./name_of_the_executable name_of_the_input_file
Parallel run script with 8 processors: 
#!/bin/ksh 
#PBS -N test
#PBS -q batch
#PBS -l walltime=12:00:00,ncpus=8
cd your_directory/
mpirun -np 8 ./name_of_the_executable name_of_the_input_file
